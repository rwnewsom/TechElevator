Questions

What is a NullReferenceException?
A null reference exception indicates that you tried to invoke a member on something pointing to a null value.
What is an ArgumentNullException?
An ArgumentNullException indicates that you passed in null to a method requiring that something not be null.
What is the purpose of a Try / Catch Block?
The try / catch block allows you to safely catch specific Exceptions that occur in the try block.
What is the CLR?
The Common Language Runtime is what runs .NET code on a specific machine. It allows the machine to execute intermediate language in order to carry out the program's behavior.
What is the difference between C# and .NET?
C# is a programming language in the .NET family of languages. .NET is the underlying platform that all .NET languages rely on. .NET provides the base class library and common language runtime that the individual programming languages rely on.
